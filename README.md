\## Multilingual Automation System

ARCOPAN implements a production-ready multilingual automation system using:
- **Polylang Free** (WordPress plugin)
- **n8n** (Automation/orchestration platform)
- **DeepL API** (Machine translation - Free tier: 500k chars/month)

### Language Configuration
- **Master language:** English (EN)
- **Target languages:** Turkish (TR), French (FR), German (DE), Russian (RU)
- **Translation flow:** All translations generated FROM English only (no chaining)

### Key Components

1. **WordPress Integration** (`arcopan-child/inc/multilingual.php`)
   - Custom REST endpoint: `/wp-json/arcopan/v1/translate`
   - Auto-trigger on post publish/update (EN only)
   - Security validation & infinite loop prevention
   - Translation status tracking

2. **n8n Workflow** (`automation/n8n/deepl-translation-workflow.json`)
   - Webhook trigger from WordPress
   - HTML cleaning & validation
   - DeepL quota checking
   - Parallel translation to 4 languages
   - WordPress post creation via REST API
   - Polylang language assignment
   - Admin notifications (success/error)

3. **Configuration**
   - `.env` file required (copy from `.env.example`)
   - DeepL API key needed
   - WordPress Application Password for REST API
   - n8n webhook configuration

### Manual Setup Steps (Required in WordPress Admin)

1. **Install & activate Polylang**
   - Go to Plugins > Add New > Search "Polylang"
   - Activate the plugin
   - Run setup wizard: enable EN, TR, FR, DE, RU

2. **Configure Translation Token**
   - Go to Tools > Multilingual Setup
   - Generate and save webhook authorization token
   - Configure auto-translate toggle

3. **Create WordPress Application Password**
   - Go to Users > Edit Your User
   - Scroll to Application Passwords
   - Create new: "n8n Automation"
   - Copy password to `.env` as `WP_API_PASSWORD`

4. **Configure n8n Workflow**
   - Install n8n (Docker or self-hosted)
   - Import workflow from `automation/n8n/deepl-translation-workflow.json`
   - Set environment variables from `.env`
   - Activate the workflow

5. **Configure Hreflang Tags** (auto-generated by Polylang)
   - Verify in page source: `<link rel="alternate" hreflang="..."`
   - Submit to Google Search Console

### Usage

**Automatic Translation:**
1. Create/edit post in English
2. Set language to English (EN)
3. Publish post
4. n8n receives webhook automatically
5. Draft posts created in TR, FR, DE, RU
6. Translator reviews and publishes

**Manual Translation Trigger:**
1. Open English post in WordPress
2. Click "Trigger Translation" button in sidebar
3. n8n processes request asynchronously
4. Check admin email for notifications

### Limits & Quotas

- **DeepL Free:** 500,000 characters/month
- **Average post:** ~5,000 characters
- **Monthly capacity:** ~30 fully translated posts (4 languages)
- **API rate limit:** 50 requests/minute

### Troubleshooting

- **Translation not triggering?** Check `.env` is loaded by WordPress
- **Posts not linking?** Verify Polylang REST API is enabled
- **Missing hreflang tags?** Clear site cache, verify in page source
- **DeepL quota exceeded?** Check monthly usage in DeepL dashboard
